🛠️ Essential Free Tools & Commands

Command-line tools for network reconnaissance, DNS queries, and network analysis

🎯

NMAP - Network Scanner

Port scanning, service detection, network reconnaissance

nmap -sV target.com
Scan ports with service versions
nmap -A -T4 target.com
Aggressive scan with OS detection
📍

NSLOOKUP - DNS Resolver

Query DNS records, resolve domains to IP addresses

nslookup google.com
Resolve domain to IP address
nslookup -type=MX google.com
Find mail exchange records
🔍

DIG - Advanced DNS Queries

Detailed DNS lookup and zone transfer attempts

dig google.com A
Query A records from DNS
dig @8.8.8.8 google.com
Query specific DNS server
📁

Dirsearch - Directory Brute Force

Find hidden directories and files on web servers

dirsearch -u http://target.com
Scan for common directories
dirsearch -u http://target.com -e php,html
Scan specific file extensions
🌐

CURL - HTTP Client

Make HTTP requests, test APIs, retrieve web content

curl http://target.com
Simple GET request
curl -I http://target.com
Show headers only
📊

Wireshark - Packet Analyzer

Capture and analyze network traffic in real-time

✓ Capture live packets on any network interface
✓ Filter by protocol (HTTP, DNS, TCP, UDP)
✓ Follow TCP streams and inspect payloads
